### 스마트폰 카메라를 이용한 실시간 노면 이슈 Detect&Report 서비스

<XX스마트폰 목업 이미지XX>

- 광주인공지능사관학교 3기 파이널프로젝트 최우수상 수상작
- 개발기간 : 2022-11.14~12.14<br/>
- 개발목표<br/>
포장도로 유지보수 조사차량이 포장도로를 조행하는 동안, 이 차량에 탑재된 스마트폰 카메라를 이용하여 포장도로의 노면내, 실시간으로 포트홀을 검출하는 시스템을 개발한다
